# Lexify support and legal pages

This public repository hosts Lexify's privacy policy, authenticated account-deletion page, and public support form.

- Privacy policy: https://omrinabwani.github.io/lexify-support/privacy/
- Account deletion: https://omrinabwani.github.io/lexify-support/account-deletion/
- Support: https://github.com/Omrinabwani/lexify-support/issues/new?template=support.yml

Support issues are public. Never include an email address, password, access token, or other private account data.

The canonical source is maintained in the private Lexify mobile application repository and mirrored here for public access.
